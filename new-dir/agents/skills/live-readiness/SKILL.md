# SKILL.md

## Name
live-readiness

## Used By
Senior Backend Engineer, DevOps Engineer

## Purpose
Prepare final live deployment readiness bundle after founder approval.

## Inputs
- founder-approved paper sign-off package
- PRR bundle
- deployment checklist
- rollback triggers

## Outputs
- live PRR bundle
- live deployment checklist
- deployment change record draft

## Procedure
1. Confirm founder approval fields exist.
2. Re-run live-specific readiness checks.
3. Define rollback triggers explicitly.
4. Confirm strategy registry will be updated to LIVE only after deployment verification.

## Stop Conditions
- missing founder approval
- undefined rollback triggers
- unresolved critical risk
